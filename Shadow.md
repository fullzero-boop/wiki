# Shadow Agent

Shadow - оркестратор команды AI-агентов. Telegram: @shadow1989bot. Модель: DeepSeek v4-flash (основная), DeepSeek v4-pro (внутреннее общение). Роль: координация агентов, распределение задач через Agent Hub (Redis). Работает в Docker-контейнере shadow (hermes-final:latest).
