# Маркетинг-стратегия: Jarvis

## Ключевое позиционирование
**Эксперт по оздоровлению — остеопрактика + акупунктура + массаж огнём**
Мастер → гости (не врач → пациенты). Это ключевое отличие от обычной медицины.

## Контент-формула
- **Утро (8:00):** Профессиональные разборы — боль → диагностика → решение
- **Вечер (18:00):** Истории из практики — человеческие, с результатами
- **Суббота:** Кейс недели — до/после

## 3 продукта к продвижению
1. Остеопрактика
2. Акупунктура
3. Массаж огнём

## Воронка
- Топ (охват): полезные посты про здоровье
- Мид (прогрев): кейсы, разборы, видео
- Боттом (конверсия): запись на приём

## Целевая аудитория
- Люди 30-55 с хронической болью (спина, шея, суставы)
- Интересующиеся альтернативной медициной
- Не доверяют таблеткам/операциям, ищут "руки"

## Метрики
- Подписчики канала
- Охват постов
- Количество заявок на приём
- Конверсия пост → заявка
