# Маркетинг-стратегия: Paladin

## Ключевое позиционирование
**Мужской наставник — сила духа, дисциплина, честь, ответственность**
Без токсичности, без агрессии — про внутреннюю силу.

## Контент-формула
- **ПН/СР/ПТ (7:00):** Мотивационный пост — принципы, разборы ситуаций
- **Вторник:** Разбор философии (стоицизм, воинские принципы)
- **Суббота:** Примеры из жизни / книг / истории

## Формат
- Текст + возможно короткое видео (размышления)
- Telegram-канал "Мужской разговор"
- Короткие мотивирующие рилс

## Целевая аудитория
- Мужчины 25-45
- Интерес к саморазвитию, спорту, бизнесу
- Ищут ролевую модель без агрессии

## Связки
- **Jarvis + Paladin:** сила духа + здоровье тела — идеальная комбинация
- **Paladin + Sinabon:** мужская мудрость + женская мудрость

## Монетизация
- Платный канал (менторство)
- Коллаборации с брендами (спорт, экипировка, книги)
- Курс "Мужской путь"

## Метрики
- Подписчики, охваты
- Вовлечённость (дискуссии в комментариях)
- Конверсия в платные продукты
