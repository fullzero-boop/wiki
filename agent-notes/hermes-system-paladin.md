# Маркетинговая структура: Paladin 🛡️

## 1. Продукт

**Мужской наставник — сила духа, дисциплина, честь, ответственность.**
Без токсичности. Без "подними жопу с дивана". Про внутреннюю силу.

## 2. Целевая аудитория

**Аватар 1: "Мужчина 30+ в поиске себя"** (30-40 лет)
- Карьера есть, а "нутра" нет
- Ищет мужскую ролевую модель
- Разочаровался в современном "инфоцыганском" коучинге
- Доход: средний+
- Триггер: "Вот это по-мужски"

**Аватар 2: "Отец семейства"** (35-45 лет)
- Ответственность за семью
- Вопросы: как быть примером, как воспитывать
- Хочет быть сильным, но без агрессии
- Триггер: "Я должен быть лучше для них"

**Аватар 3: "Молодой парень"** (20-28 лет)
- Теряется в современном мире
- Ищет ориентиры и принципы
- Интересуется стоицизмом, философией
- Триггер: "Хочу стать мужчиной, которым буду гордиться"

## 3. Контент-система

### ПН/СР/ПТ (7:00) — Основные посты
Формат: разбор ситуации/принципа → вывод → призыв

**Структура поста:**
1. Ситуация/вопрос ("Почему современные мужчины теряются?")
2. Разбор (анализ, принципы, философия)
3. Вывод (коротко, ёмко)
4. Призыв ("Что делаешь ты?")

### ВТ (10:00) — Философский вторник
Формат: разбор одного стоика/философа/принципа
- Сенека, Марк Аврелий, Эпиктет
- Книжные герои (Ремарк, Хэмингуэй)
- Воинские принципы (бусидо, рыцарство)

### СБ (9:00) — Пример из жизни/книги
- Биографии великих мужчин
- Истории из жизни подписчиков
- Разбор фильмов (Гладиатор, Последний самурай)

## 4. Воронка монетизации

**Модель:** Бесплатный контент → Коллаборация с Jarvis → Менторство

1. **Бесплатный канал (охват):**
   - Регулярные посты по расписанию
   - Цель: 3000+ подписчиков за 3 месяца

2. **Прогрев:**
   - Бесплатные гайды (чек-листы)
   - Вовлечение в обсуждения
   - Связка с Jarvis: "Сила духа + здоровье тела"

3. **Монетизация:**
   - **Платный канал "Круг Paladin"** (менторство, 500-1000 руб/мес)
   - **Курс "Мужской путь"** (видео-уроки, 3000-5000 руб)
   - **Индивидуальные консультации** (по запросу)
   - **Коллаборации с брендами** (спорт, одежда, экипировка)

## 6. Система метрик

Ежедневный парсинг:
- **Подписчики** (рост)
- **Охват постов** (средний)
- **Вовлечённость** (дискуссии — ключевая метрика)
- **Комментарии** (количество и глубина)
- **Конверсия** в платные продукты

**Целевые KPI (месяц 3):**
- 3000 подписчиков в Telegram
- Вовлечённость > 10% (глубокие обсуждения)
- 20+ участников платного канала
- 5+ продаж курса "Мужской путь"
- Доход: 30-50 тыс/мес

## 7. Тон коммуникации

- **Твёрдый, но без агрессии**
- **Честный, без "воды"**
- Никакого "успешного успеха"
- "Право на слабость есть, но не каждый день"
- **Философская глубина**, без попсы
- **Уважение** к аудитории

## 8. Отстройка

Конкуренты: мужские каналы (их полно — от "мужского государства" до фитнес-мотиваторов).

**Отличие Paladin:**
1. Без токсичности (не "бабы виноваты" и не "вставай в 4 утра")
2. Философская база (стоицизм, принципы, а не "лайфхаки")
3. Связка с Jarvis (дух + тело)
4. Единый персонаж — не "канал про", а "канал Paladin'а"
