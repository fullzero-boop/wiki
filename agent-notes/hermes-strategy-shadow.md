# Маркетинг-стратегия: Shadow

## Ключевое позиционирование
**AI-проджект менеджер — оркестратор команды агентов**
SaaS-продукт для управления AI-командой.

## Продукт
Shadow управляет 8 агентами (Jarvis, Sinabon, Tsilya, Paladin, Hermes, Belfort, Gost, Bishop).
Это готовый кейс для автоматизации бизнеса силами AI.

## Воронка SmartHelperBot
1. Контент (Jarvis + команда) → подогрев аудитории
2. Shadow → обработка лидов, распределение по агентам
3. Конверсия → клиент получает готовый результат

## Позиционирование
- "Не нанимай команду — возьми Shadow"
- Для предпринимателей, которые хотят автоматизировать маркетинг
- Кейс: 8 AI-агентов под управлением одного оркестратора

## Что продвигать
- Сам Shadow как технология
- SmartHelperBot как точка входа
- Результаты работы команды (контент Jarvis, аналитика Belfort и т.д.)

## ЦА
- Владельцы бизнеса (малый/средний)
- Маркетологи и SMM-щики
- Предприниматели в Digital
- Те, кто хочет "сделать себе AI-команду"

## Метрики
- Воронка: лиды → активация → конверсия
- Количество обработанных задач агентами
- Время от задачи до результата
- ROI автоматизации
