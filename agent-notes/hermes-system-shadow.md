# Маркетинговая структура: Shadow 👤

## 1. Продукт

**AI-проджект менеджер.** 
Shadow управляет командой AI-агентов. Это SaaS-продукт для предпринимателей: "Не нанимай команду — возьми Shadow".

## 2. Целевая аудитория

**Аватар 1: "Предприниматель в Digital"** (30-45 лет)
- Имеет бизнес в онлайне (инфобиз, маркетинг, SaaS)
- Нужен контент, но нет команды
- Пробовал нанимать — дорого, неэффективно
- Доход: высокий
- Триггер: "Я плачу людям, а они сливают дедлайны"

**Аватар 2: "Маркетолог-одиночка"** (25-35 лет)
- Тащит всё на себе
- Нужен ассистент, контент-мейкер, аналитик
- Не может нанять команду
- Триггер: "Мне нужна команда, но бюджет ограничен"

**Аватар 3: "Владелец агентства"** (35-50 лет)
- 10-50 сотрудников
- Хочет автоматизировать маркетинг AI
- Ищет competitive advantage
- Триггер: "Мои конкуренты уже используют AI, а я нет"

## 3. Продуктовая матрица Shadow

### Бесплатный слой (SmartHelperBot)
- Входная точка для всех клиентов
- Диагностика задачи ("Что нужно сделать?")
- Распределение по агентам (Jarvis, Belfort и т.д.)
- Демо: "Попробуй — получи результат"

### Платные тарифы
| Тариф | Функции | Цена |
|-------|---------|------|
| **Start** | 1 агент (Jarvis), 10 запросов/мес | $29/мес |
| **Team** | 3 агента (Jarvis + Sinabon/Циля + Belfort), 50 запросов | $99/мес |
| **Studio** | Все 5 контент-агентов, 200 запросов, приоритет | $299/мес |
| **Enterprise** | Полный доступ + кастомные агенты | $999+/мес |

## 4. Воронка

```
ХОЛОД → SmartHelperBot → Shadow → РЕЗУЛЬТАТ
```

1. **Холод** — контент Jarvis/Sinabon/Tsilya/Paladin/Belfort привлекает аудиторию
2. **SmartHelperBot** — ловит лида, диагностирует задачу
3. **Shadow** — распределяет задачу агенту
4. **Результат** — клиент получает контент/аналитику
5. **Конверсия** — клиенту нравится → он подписывается на тариф

## 5. Контент-система для Shadow

### Форматы:
- **Кейсы:** "Как мы сделали контент-план на месяц за 2 часа"
- **Сравнения:** "Shadow vs команда из 5 человек"
- **Технические разборы:** "Как работает оркестрация AI-агентов"
- **Live-демо:** "Смотри, как Shadow решает задачу в реальном времени"

### Каналы:
- Telegram-канал Shadow
- SmartHelperBot (точка входа)
- YouTube (демо, разборы)
- VC.ru / Habr (статьи)

## 6. Система метрик

- **Лиды = запуски SmartHelperBot**
- **Конверсия = подписка / лиды**
- **Churn = отписки / месяц**
- **ARR = Monthly Recurring Revenue × 12**
- **LTV = средний чек × среднее время жизни клиента**
- **CAC = cost per acquisition**

**Целевые KPI (6 месяцев):**
- 1000+ запусков SmartHelperBot/мес
- 50+ платных подписчиков
- Конверсия 5-10%
- ARR $30-100K

## 7. Отстройка

Конкуренты: WriteSonic, Jasper, Copy.ai — но это просто генераторы текста.

**Отличие Shadow:**
1. **Оркестрация**, а не генерация — Shadow управляет **командой агентов**
2. Каждый агент — специалист (Jarvis = здоровье, Belfort = крипто)
3. **SmartHelperBot** — входная точка, а не просто сайт
4. Реальный рабочий продукт, а не "ещё один AI-писатель"
