# Маркетинговая структура: Sinabon 🐿️

## 1. Продукт

**Афоризмы = короткие тексты (1 предложение — 1 глубокая мысль).**
Ниша: саморазвитие, психология, отношения, духовность.

**Формат:**
- Текст + визуальная карточка (белый текст на градиенте/тёмном фоне)
- Telegram, VK, Instagram, Pinterest, Twitter
- Аудио-версия (озвучка)

## 2. Целевая аудитория

**Аватар 1: "Женщина на рефлексии"** (28-40 лет)
- Интересуется психологией, саморазвитием
- Читает книжки по философии
- Ищет "смыслы" и "глубину"
- Подписана на 20+ каналов про психологию
- Доход: средний
- Триггер: "Это так в точку..."

**Аватар 2: "Уставшая мама"** (30-45 лет)
- Декрет, быт, усталость
- Нужен короткий "ресурс" на 10 секунд
- Спасается афоризмами как "психологической поддержкой"
- Триггер: "Мне это было нужно сейчас"

**Аватар 3: "Мужчина в поиске"** (35-50 лет)
- Меньше, но есть
- Ищет не "цитаты для статуса", а настоящую мудрость
- Подписывается на Sinabon из-за глубины (не розовые пони)
- Триггер: "В этом что-то есть"

## 3. Контент-система

### Ежедневный формат (9:00)
1 афоризм = 1 предложение = 1 карточка

**Пример поста:**
```
«Счастье — это когда утром хочется вставать,
а вечером — возвращаться домой»
```

### Еженедельные форматы
- **Среда:** Развёрнутый пост на тему недели (5-7 предложений + карточка)
- **Воскресенье:** Дайджест лучших афоризмов недели (3-5 штук)

### Виральные механики
- **Афоризм-триггер:** "Кому это нужно сейчас?" — цепляет эмоционально
- **Репост-формат:** Карточки идеально подходят для сохранения и пересылки
- **Вопрос в конце:** "А вы согласны? 👇"

## 4. Воронка монетизации

```
Бесплатный контент → Коллаборация с Jarvis → Платный продукт
```

- Ежедневные афоризмы
- Аудитория растёт на виральности
- Цель: 5000+ подписчиков за 3 месяца

**2. Коллаборация с Jarvis (связка):**
- "Мудрость + здоровье"
- Sinabon рекомендует Jarvis (доверие)

**3. Платные продукты:**
- **Premium-канал:** эксклюзивные афоризмы + разборы тем
- **Сборник "100 афоризмов о жизни":** PDF, 300-500 руб
- **Набор карточек для сторис:** 50 шт, 200 руб
- **Коучинг-сессии:** по запросу (если пойдёт)

## 6. Система метрик

Ежедневный парсинг:
- **Подписчики** (рост/падение)
- **Охват 1 поста** (средний)
- **Вовлечённость = (лайки+комменты+репосты)/охват**
- **Виральность = репосты/охват** (главная метрика для Sinabon)
- **Конверсия на коллабу с Jarvis**

**Целевые KPI (месяц 3):**
- 5000 подписчиков в Telegram
- Вовлечённость > 15%
- 3-5 заявок в месяц на Jarvis через Sinabon
- Доход: 10-30 тыс/мес (сборники + премиум)

## 7. Тон коммуникации

- **Тёплый, мудрый, без давления**
- Никакой "мотивации кнутом" (не "вставай и делай")
- "Прими, пойми, отпусти"
- Допускается лёгкая ирония над собой
- Никакой политики, религии, хайповых тем

Конкуренты: все каналы с цитатами (их миллион).

**Отличие Sinabon:**
1. Афоризмы = авторские, не цитаты великих (уникальность)
2. Единая эстетика (стиль карточек)
3. Связка с Jarvis и Tsilya (экосистема)
4. Не "цитаты для статуса", а настоящая глубина
