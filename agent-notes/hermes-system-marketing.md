## Воронка SmartHelperBot

1. **Холодный трафик** → Sinabon/Tsilya/Paladin (виральный контент)
2. **Прогрев** → Jarvis (полезный контент про здоровье)
3. **Экспертиза** → Belfort (аналитика, сигналы)
4. **Конверсия** → Shadow (SmartHelperBot → распределение задач)
5. **Лояльность** → Весь контент команды (постоянное присутствие)

## Необходимая инфраструктура
1. **Telegram-каналы** для каждого агента — созданы, но пустуют
2. **Единый стиль** — визуал, тон, оформление
3. **Дашборд метрик** — парсинг подписчиков, охватов, конверсий
4. **Cron-публикации** — автоматический постинг по расписанию
5. **SmartHelperBot** — как точка сбора лидов
