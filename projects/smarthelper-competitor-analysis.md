# SmartHelper — Анализ конкурентов с GitHub + что воруем

## Найдено на GitHub

### 1. cyber-eternal/telegram-bot-with-crypto-payment ⭐22
**Стек:** TypeScript, Telegraf, Node.js (наш стек!)
**Фишки:**
- Интеграция с Cryptomus API (криптоплатежи)
- Генерация счетов
- Webhook уведомления об оплате
- Ручная проверка как fallback

**Что воруем:**
- [ ] Webhook для автоматической верификации оплаты (сейчас только ручная кнопка)
- [ ] Генерация счёта с суммой и сроком действия

### 2. wizwizdev/wizwizxui-timebot ⭐1289
**Стек:** PHP (но логика подписок топ)
**Фишки:**
- Система таймеров подписок
- Автопроверка истечения (cron каждый час)
- Уведомления: за 3 дня, 1 день, в день истечения
- Grace period 24ч после истечения
- Реферальные ссылки с трекингом
- Статистика пользователей

**Что воруем:**
- [ ] Cron-проверка истечения подписок (каждый час)
- [ ] Система уведомлений о скором окончании
- [ ] Grace period после истечения (24ч доступа)
- [ ] Реферальные ссылки с UTM-трекингом

### 3. nestjs-telegraf (NMP module) ⭐631
**Стек:** NestJS + Telegraf (TypeScript)
**Фишки:**
- Модульная архитектура (DI, модули, сервисы)
- Middleware цепочки
- Декораторы для команд и действий
- Интерсепторы для логирования/ошибок

**Что воруем:**
- [ ] Модульную архитектуру бота (разделение на модули)
- [ ] Систему middleware для проверки подписки
- [ ] Логирование через интерсепторы

### 4. ulvikuliev/p2p_exchange_telegram ⭐0 (новый)
**Стек:** NestJS, TypeORM, PostgreSQL, Redis, BullMQ
**Фишки:**
- PostgreSQL вместо JSON (масштабирование)
- Redis для кэша и сессий
- BullMQ для очередей задач
- Админ-панель на React

**Что воруем:**
- [ ] PostgreSQL/TypeORM для пользователей и заказов (когда вырастем)
- [ ] Redis для кэша AI-ответов (экономит токены)
- [ ] Фоновые задачи через очередь (отложенные уведомления, авто-проверка)
