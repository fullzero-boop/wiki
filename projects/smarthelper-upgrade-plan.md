### 🧠 Новая логика работы бота

```
1. /start (или реферальная ссылка ref_xxx)
    │
    ├── Профиль есть? → показываем меню
    │
    └── Нет профиля → предлагаем опросник
         │
         ├── Пользователь отвечает на 7 вопросов
         │   ├── Ниша
         │   ├── Платформы
         │   ├── Стиль контента
         │   ├── Потребности
         │   ├── Частота
         │   ├── Аудитория
         │   └── Боли
         │
         └── AI генерирует персонализированное предложение
              ├── "Вот что я могу для ТВОЕГО бизнеса"
              ├── Рекомендуемый тариф
              └── Кнопки: Попробовать / Сразу купить / Вопрос
```

### 💎 После оплаты

```
Оплата → 
├── Grace period 15 мин (доступ сразу, TronGrid проверяет фоном)
├── Поздравление + предложение заполнить профиль
└── Через 30 сек → проактивное предложение #1
    Через 60 сек → проактивное предложение #2
    Через 90 сек → проактивное предложение #3
```

## 🔧 ЧТО НУЖНО ДЛЯ ЗАПУСКА

1. ✅ Ссылка на GitHub (код бота)
2. Установить модули в проект (npm i)
3. Подключить модули в bot.js
4. Запустить

**Интеграция в bot.js займёт 15 минут:**
```javascript
const { registerModules } = require('./modules/bot-core');
const config = { botUsername: 'SmartHelper579Bot', adminId: ADMIN_ID };
registerModules(bot, config);
```
