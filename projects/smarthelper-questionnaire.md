# SmartHelper — Опросник (5-7 вопросов)

## Цель
После опросника AI знает о бизнесе клиента ВСЁ → выдаёт персонализированное предложение → конверсия ×3

---

## 📋 ВОПРОСЫ

### Вопрос 1 — Ниша (текст/кнопки)
```
🧭 Расскажи про свой бизнес

Кто ты и чем занимаешься? Выбери или напиши своё:

🔹 Салон красоты / Бьюти
🔹 Фитнес / Спорт / ЗОЖ
🔹 Психология / Коучинг
🔹 Интернет-магазин
🔹 Инфобизнес / Обучение
🔹 Юридические / Бухгалтерские услуги
🔹 Ресторан / Кафе
🔹 Другое (напишу сам)
```

*Если выбрал "Другое" → предлагает ввести текстом*

### Вопрос 2 — Платформы (кнопки, множественный выбор)
```
🌐 Где ты ведёшь соцсети?

Выбери площадки (можно несколько):

✅ Telegram — канал/чат
✅ VK — сообщество/страница
✅ Дзен — блог/канал
✅ Instagram
✅ Другое
❌ Пока нигде, но планирую
```

### Вопрос 3 — Стиль контента (кнопки, один выбор)
```
🎨 Какой стиль постов тебе ближе?

🔹 Экспертный — глубокие разборы, факты, исследования
🔹 Душевный — истории, личный опыт, забота
🔹 Продающий — акции, товары, кейсы
🔹 Развлекательный — юмор, мемы, лайфстайл
🔹 Смешанный — всего понемногу
```

### Вопрос 4 — Потребности (кнопки, множественный выбор)
```
🎯 Что нужно твоему бизнесу прямо сейчас?

✅ Посты и контент — регулярные публикации
✅ Контент-план — стратегия на месяц
✅ Прогрев аудитории — продающие серии постов
✅ Аналитика — что заходит, что нет
✅ Автоматизация — чтобы само постилось
✅ Всё сразу — хочу полное ведение
```

### Вопрос 5 — Частота (кнопки)
```
📅 Как часто хочешь публиковать?

🔹 1-2 раза в неделю (лёгкий режим)
🔹 3-5 раз в неделю (активный)
🔹 Каждый день (максимум)
🔹 Не знаю, нужна стратегия
```

### Вопрос 6 — ЦА (текст)
```
👥 Кто твоя аудитория?

Опиши коротко: возраст, пол, интересы, боли.

Например: «Женщины 30-45, мамы в декрете, хотят выглядеть хорошо, но нет времени на спортзал»
```

### Вопрос 7 — Боли (текст)
```
💡 Что сейчас не устраивает в ведении соцсетей?

Напиши пару слов:
❌ Нет времени
❌ Не знаю что писать
❌ Нет результатов от постов
❌ Другое
```
